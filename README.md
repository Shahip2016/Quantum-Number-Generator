# Quantum Random Number Generator (QRNG)

This repository implements a high-performance Quantum Random Number Generator (QRNG) based on 2025 research advancements.

## Key Features
- **Vacuum Fluctuation Simulation**: Modeling homodyne detection of quantum states.
- **ZCA Whitening**: Advanced post-processing to eliminate correlations.
- **NIST Verification**: Built-in statistical analysis for randomness certification.
